# trade-alpha

股票预测程序

## 功能

- [x] 数据层：Tushare 数据获取，MongoDB 存储
- [x] 分析层：技术指标计算（MA、MACD、RSI、ATR、OBV、KDJ、BOLL、BIAS等）
- [x] 预测层：价格预测（XGBoost、LSTM）
- [x] 训练层：样本混合训练、模型持久化
- [x] 策略层：交易信号生成（ATR 动态追踪止损、评分下滑过滤、每日买入上限控制）
- [x] 账户层：资金管理、交易记录
- [x] 回测层：策略回测、指标计算、周度候选股票池（市值+涨幅双选+滚动留存）
- [x] 市场分析层：MA10/MA60 基线分析、市场阶段检测、排名留存率
- [x] 数据分析层：特征统计、直方图、箱线图、缺失值分析
- [x] 实盘建议层：每日全市场评分排名、买卖建议生成
- [x] 排名优化：趋势加减分、动量加成/扣分、暴涨排除、评分下滑过滤、EWMA 平滑
- [x] 股票管理：行业筛选、历史日期回溯、回测参与开关（单只/批量）
- [x] API 层：FastAPI RESTful 接口
- [x] 前端界面：Vue 3 + Vuetify 4
- [x] 日志模块：结构化日志，请求链路追踪
- [x] 异步任务管理：训练、回测、数据分析异步执行

## 使用指南

通过前端 Web 界面（http://localhost:3000）完成完整的股票分析和回测流程：

### 1. 数据管理 `/data`

启动后自动从 Tushare 同步 A 股股票列表和数据。如需手动操作：
- **更新股票列表**：点击"更新股票列表"获取最新 A 股数据
- **行业筛选**：通过行业下拉框筛选指定行业的股票
- **历史日期回溯**：输入历史日期查看该日期存在的股票列表
- **回测参与开关**：单只或批量切换股票的 `is_active_for_backtest` 状态，控制是否参与回测
- **下载数据**：选中股票，设置日期范围，点击下载
- **查看 K 线**：点击股票行查看 K 线图
- **删除数据**：选中不需要的股票数据删除

### 2. 数据分析 `/data-analysis`

对已下载的股票数据做特征分析：
- **发起分析任务**：选择股票范围（按市值排名或指定代码）、时间范围和特征字段
- **查看结果**：任务完成后查看统计指标（均值/标准差/分位数/缺失率）、直方图、箱线图
- **管理**：查看历史结果列表，支持删除不需要的结果

### 3. 账户配置 `/account-configs`

配置回测使用的账户参数：
- 设置初始资金、买入费率、卖出费率、印花税率、最低手续费
- 支持多个账户配置，回测时选择使用

### 4. 策略配置 `/strategies`

配置回测使用的交易策略：
- 创建不同策略类型（组合策略 / 单股票策略）
- 动态设置策略参数（评分阈值、止损、持仓限制等）
- 回测时选择对应的策略配置

### 5. 模型配置 `/models`

配置预测模型参数：
- 支持 **XGBoost** 和 **LSTM** 两种模型类型
- 配置特征字段、分类预测周期（3日/5日）、涨跌分类阈值
- 根据模型类型动态调整参数（LSTM 需设置序列长度）
- 配置完成后点击"训练"跳转至训练页

### 6. 训练管理 `/trainings/manage`

发起训练任务：
- 选择已创建的模型配置
- 选择训练股票（通过市值排名范围，默认 1-1000 只）
- 设置训练时间范围和名称
- 提交后实时查看任务进度

### 7. 训练记录 `/trainings/records`

查看和管理训练结果：
- 按模型配置筛选训练历史
- 查看训练指标（样本数、准确率、特征重要性、类别分布）
- **LSTM 模型**额外显示 Final Loss 和每 Epoch Loss
- 点击"预测"使用训练好的模型对指定股票进行预测
- 查看预测结果（各周期的预测类别和上涨概率）

### 8. 回测管理 `/backtest/manage`

发起回测任务：
- 选择账户配置、训练结果、策略配置
- 自动推断策略模式（组合策略 / 单股票策略）
- 设置时间范围和回测名称
- 组合模式设置计算范围、市值前 N、涨幅前 N（用于周度候选股票池双选+滚动留存）
- 单股票模式输入股票代码

### 9. 回测记录 `/backtest/records`

查看回测结果：
- 总收益率、夏普比率、最大回撤、波动率、胜率等核心指标
- 基线对比（买入持有策略的收益率和最大回撤）
- 超额收益分析
- 查看详细交易记录

### 10. 交易记录 `/backtest/trades`

查看所有交易流水：
- 按账户、策略、训练、股票多维度筛选
- 每笔交易的买卖方向、价格、股数、手续费、评分等信息
